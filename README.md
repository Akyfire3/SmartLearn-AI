# SmartLearn AI

AI-powered study assistant that transforms PDF notes into smart learning resources.

## Features

* PDF Upload & Processing
* AI Generated Summary
* Revision Notes Generation
* Automatic Topic Extraction
* MCQ Quiz Generation
* Topic-wise Quiz Generation
* Multiple Difficulty Levels (Easy, Medium, Hard, Mixed)
* Exam Questions (2, 5, and 10 Marks)
* Viva Questions with Study & Practice Modes
* Copy to Clipboard Support
* Topic-based Learning Resources

## Tech Stack

* React.js
* Vite
* Tailwind CSS
* OpenRouter API
* JavaScript

## Installation

```bash
git clone https://github.com/Akyfire3/SmartLearn-AI.git
cd SmartLearn-AI
npm install
npm run dev
```

## Environment Variables

Create a `.env` file in the root directory:

```env
VITE_OPENROUTER_API_KEY=YOUR_OPENROUTER_API_KEY
```
